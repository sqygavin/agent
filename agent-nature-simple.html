01

Agent 是什么？

像介绍一个新同事一样，把 Agent 讲清楚

💡 一句话定义

Agent = 能干活的"数字员工"

想象你招了一个特别聪明的实习生。这个实习生：

  • 1
    能看懂你的话
    你说"帮我订明天去上海的机票"，他听得懂
  • 2
    会自己想办法
    他会去查航班、比价、选时间
  • 3
    会记住事情
    上次你说过不坐红眼航班，他记得
  • 4
    会动手干活
    真的帮你把票订了，而不是只说"建议你去官网看看"
核心洞察
Agent 不是"更聪明的搜索框"，而是一个能自主行动的助手。

普通 AI

你："帮我订机票"
它："建议您访问携程官网~"
→

Agent

你："帮我订机票"
它：直接帮你订好
"已出票，下午2点起飞"
02

Agent 的三板斧

会想、会记、会用工具

💭

会想（Planning）

做事前，先想想怎么做。像考试时写解题步骤，而不是瞎蒙答案。

Planning

进阶：ReAct = 边做边想

🗂️

会记（Memory）

有记性。记得上次你说的话、你的偏好、之前犯过的错。

Memory
🔧

会用工具（Tool Use）

会用工具。Agent 能调用 API、发邮件、写代码——真正帮人类完成任务。

Tool Use
为什么"会用工具"最重要？
不会用工具 = 只能动嘴：能说话、给建议，但做不了实事。

会用工具 = 能动手：能订票、发邮件，写代码——真正帮人类完成任务。

所以 Tool Use 是 Agent 和普通 AI 的本质区别。

03

落地的本质

一个"靠谱"问题

"Agent 最大的敌人，不是不会做，而是做错。"

— 来自踩坑无数的工程师们

📊 为什么"靠谱"这么难？

Agent 做事像一条流水线，每一步都可能出错。

步骤 1
理解任务
→
步骤 2
查资料
→
步骤 3
做决策
→
步骤 4
执行
算一算
每步正确率 90%
4步后：90% × 90% × 90% × 90% = 66%
1/3 的概率会出错！
核心洞察
可靠性是 Agent 落地的第一道坎。

不是"能不能做"，而是"做不做得好"。
每多一步，就多一分出错的风险。
好的 Agent，要像好的员工一样——稳，比聪明更重要。

🛡️ 怎么让 Agent 更靠谱？

  • 1
    给个"工作手册"（Schema 约束）
    就像新人入职有 SOP，告诉 Agent 什么能做、什么不能做、怎么做是对的。
  • 2
    让它自己检查（自我修正）
    就像写完报告自己检查一遍，发现错了就改。
  • 3
    做事要有证据（引用验证）
    就像论文要引用来源。Agent 说的每句话，都要能追溯到证据。
  • 4
    关键决策要人把关（Human-in-the-loop）
    就像大额支出需要领导审批，Agent 在关键步骤需要人确认。
04

记忆：Agent 的灵魂

没有记忆，就没有 Agent

👤 人的记忆

  • • 短期记忆：当前对话记住
  • • 长期记忆：以前的事也记得
  • • 碎片化：会忘、会混

🤖 Agent 的记忆

  • • 上下文：当前对话内
  • • 向量数据库：像超级笔记
  • • 可配置：想记什么记什么
核心洞察
没有记忆，就没有 Agent。

就像一个失忆的人，每次见面都问"你是谁"——
没有记忆的 Agent，就是一个每次都要从头开始的"新手"。

Agent 记忆的六种形态（OWASP 框架）

KC4.1 这个对话里我说过的话
KC4.2 跨 Agent 的共同记忆（像共享笔记本）
KC4.3 上次聊过的事，下次还记得
KC4.4 所有 Agent 共享的长期记忆
KC4.5 记得我是谁、喜欢什么
KC4.6 记得所有用户的信息
05

七句话，说清楚 Agent

核心洞察总结

  • 1
    Agent 不是更聪明的 AI，而是能自己干活的 AI
    关键词：自主行动
  • 2
    记忆是 Agent 的灵魂，没有记忆就是"失忆症患者"
    关键词：上下文连续性
  • 3
    ReAct = 边想边做，不是想好了再动手
    想一点、做一点、调整一点。关键词：动态适应
  • 4
    幻觉是 Agent 的最大敌人，就是"一本正经地胡说八道"
    关键词：真实性验证
  • 5
    可靠性是落地的前提，不稳的 Agent 还不如不用
    关键词：系统工程
  • 6
    渐进式演进是关键，不可能一步登天
    关键词：从 LLM 到 Agent 的演进路径
  • 7
    人机协作是安全底线，Agent 应该增强人类而非替代
    关键词：Human-in-the-loop

一句话总结

Agent = 会想 + 会记 + 会用工具 + 靠谱
就像一个靠谱的助理：
你交代任务，它自己去想办法、查资料、动手做。
做完了告诉你结果，做错了自己修正。

它不是更聪明的搜索框，而是一个能自主行动的助手。

基于 Cameron R. Wolfe、Oracle、OWASP GenAI 等来源的深度研究
